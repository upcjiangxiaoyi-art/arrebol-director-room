# Arrebol Director Room 红霞导演室

SillyTavern RP 场外导演插件。副 API 读取最近 N 轮剧情，输出情感方向，注入正文供主模型（如 Gemini）执行。

> 设计理念：长上下文给导演，短方向给演员。给”在场感极强但容易发疯”的主模型配一个理智导演。

## v0.1 功能（单导演手动台）

- 选择复盘范围：最近 10 / 20 / 30 / 50 / 自定义轮
- 手动补充框：填角色卡要点、世界书、当前担心
- 情感导演预设框：自定义分析方法（内置默认预设）
- 调副 API（OpenAI 兼容 / OpenRouter）
- 预览区：可手动编辑 / 带补充指令重新分析 / 复制 / 注入正文末尾
- 聊天框旁可拖拽的快捷入口

## 安装

扩展面板 → Install extension → 填仓库地址。装完刷新页面。

## 使用

1. 点聊天框旁的 🎬 快捷入口打开面板
1. 「配置」区填 API（OpenRouter 填 `https://openrouter.ai/api/v1`）、加载模型、选模型
1. 选复盘轮数
1. 「手动补充」填角色和当前剧情背景
1. 「情感导演预设」可改可不改（有默认）
1. 点「生成方向」，结果出现在「导演方向」区
1. 不满意可手动改，或写补充指令点「重新分析」
1. 满意后点「注入正文末尾」，下一轮主模型就能读到
